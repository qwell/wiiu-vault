# WiiU Vault

WiiU Vault is a multi-platform open source Wii U ROM manager.

## Download

Download the latest release from GitHub Releases:

https://github.com/qwell/wiiu-vault/releases

## Features

- Manage and organize a Wii U game library
- Search and filter titles
- Track title status
- Download and verify title metadata
- Runs on Windows, macOS, and Linux

## Installation

Download the ZIP for your platform, extract it, and run:

- Windows: `wiiu-vault.exe`
- macOS/Linux: `./wiiu-vault`

The packaged launcher downloads and verifies a Node.js runtime on first run.
